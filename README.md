# Diffy

A quick shortcut to publish your working tree to diffy.org.

## Features

This extension adds a new command: `Share diff`. When used in a workspace with an initialized repository, it uploads your diff to [Diffy.org](https://diffy.org/) and then prompts you to either open the diff in a browser or copy a URL to your clipboard.
